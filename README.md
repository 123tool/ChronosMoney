## ChronosMoney // Professional Time-to-Money Calculator

ChronosMoney adalah aplikasi web kalkulator analitik premium yang dirancang secara profesional untuk membantu pengguna menghitung konversi nilai intrinsik waktu kerja nyata yang dibutuhkan untuk dapat membeli aset produk atau layanan tertentu. Aplikasi ini dikembangkan dengan performa tinggi, zero dependencies framework luar, aman dari injeksi XSS, serta dilengkapi persistensi data terenkripsi lokal otomatis (*Sandboxed LocalStorage States*).

---

## Fitur Utama

* **Real-time Dynamic Chronos Math Engine**: Menghitung jam kerja kotor (*Gross Hours*) hingga jam kerja bersih (*Net Working Hours*) secara akurat setelah dikurangi porsi estimasi pajak pendapatan.
* **Contextual Multi-Currency Engine Suite**: Mendukung multi-mata uang dinamis (IDR, USD, EUR, SGD) lengkap dengan penyesuaian otomatis simbol mata uang serta pelokalan representasi format angka internasional.
* **Standard Work Shift Equivalent Visualizer**: Mengonversi jam kerja ke dalam satuan metrik *Shift Kerja Standard* (basis 8-jam kerja konvensional per hari) untuk pemahaman psikologis nilai uang yang lebih baik.
* **Persistent Historical Ledger Table**: Menyimpan riwayat pencarian kalkulasi Anda secara aman di dalam penyimpanan browser lokal tanpa perlu adanya server eksternal ataupun registrasi akun.
* **Data Serialization Export Feature**: Ekspor seluruh rekaman data riwayat kalkulasi langsung ke dalam format berkas `.csv` clean structured untuk kebutuhan analisis eksternal (Spreadsheet / Microsoft Excel).
* **Bi-directional Premium UI Dark Mode Theme Switcher**: Antarmuka kelas profesional dengan *smooth transitions animation rendering pipeline* yang beradaptasi dengan preferensi sistem operasi pengguna.
